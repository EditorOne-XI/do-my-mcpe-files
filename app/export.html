Exporting Minecraft Files

This feature was added to easily export all of your Minecraft data, essentially creating a backup of it before updating or reinstalling the game, as both actions can cause issues. Updating the game might cause issues or even corrupt your worlds when they are loaded, reinstalling the game may result in the loss of all your data because of changes to how Minecraft stores files after Android 11 (API level 30) enforced the Scoped Storage feature which Minecraft later adopted in the Caves and Cliffs: Part II update.

  • Exported files will be stored in the Download directory.
  • The export will be canceled if the com.mojang directory does not exist.
  • Exporting is possible as long as the app contains a com.mojang directory, even if an error occurs. directory even when an error occurs.
  • The exported archive file has a specific signature within. Modifying it will cause it to be rejected by the Import Builder.
